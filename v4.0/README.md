# WhatsApp Bot v4.0 🤖

最後更新：2026-04-04

## ✨ 主要功能

### 🔄 新指令系統（v4.2）
- `!` 開頭 = 機器人指令
- `#` 開頭 = 豆包 AI 指令
- 數字指令 `!1` ~ `!4` 快速啟動工具

### 🤖 豆包 AI 整合
- 透過 OpenCLI 連接豆包 AI
- 支援長篇回應（60秒超時）
- 群組和私聊都可用

### 👥 群組與私聊支援
- 群組和私聊都可以使用
- 獨立的授權管理

### 🛠️ CLI 管理工具
- 命令列管理界面
- 查看狀態、工具列表
- 便捷的除錯功能

## 📁 檔案結構

```
v4.0/
├── index.js              # 主程式
├── cli.js                # CLI 管理工具
├── package.json          # 依賴配置
└── *.html                # 工作流程 UI
```

## 🔧 引用工具版本

v4.0 使用 `tools/` 目錄：
- `../tools/doubao-ai/r1`
- `../tools/weather/r2`
- `../tools/msg-extractor/r2`

## 🚀 使用方式

### 啟動機器人
```bash
node v4.0/index.js
```

### 使用 CLI 工具
```bash
node v4.0/cli.js status
node v4.0/cli.js tools
```

## 📝 可用指令

| 指令 | 功能 | 場景 |
|------|------|------|
| `!start` | 顯示工具選單 | 群組/私聊 |
| `!stop` | 停止工具 | 群組/私聊 |
| `!狀態` | 查看狀態 | 群組/私聊 |
| `!1` ~ `!3` | 啟動對應工具 | 群組/私聊 |
| `#問題` | 問豆包 AI | 群組/私聊 |

### 工具列表
1. **doubao-ai** - 豆包 AI 整合
2. **msg-extractor** - 訊息提取
3. **weather** - 天氣查詢
