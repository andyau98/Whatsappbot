# WhatsApp Bot v4.0 🤖

最後更新：2026-04-03

## ✨ 主要功能

### 🔄 滑動回覆支援
- 支援滑動（引用）回覆觸發指令
- 智能識別回覆對象（自己/他人）
- 群組和私聊不同處理邏輯

### 🧠 智能指令解析
- 自然語言操作，無需記憶固定指令
- 自動識別天氣查詢等意圖
- 支援多種同義詞和表達方式

### 👥 群組與私聊區分
- 群組需要 @機器人 才能觸發
- 私聊直接發送指令
- 獨立的授權管理

### 🛠️ CLI 管理工具
- 命令列管理界面
- 查看狀態、工具列表
- 便捷的除錯功能

## 📁 檔案結構

```
v4.0/
├── index.js              # 主程式
├── cli.js                # CLI 管理工具
├── package.json          # 依賴配置
├── tools/                # 工具引用（實際在 ../tools/r2/）
└── *.html                # 工作流程 UI
```

## 🔧 引用工具版本

v4.0 使用 `tools/r2/` 版本：
- `../tools/weather/r2`
- `../tools/smart-command/r2`
- `../tools/msg-extractor/r2`

## 🚀 使用方式

### 啟動機器人
```bash
node v4.0/index.js
```

### 使用 CLI 工具
```bash
node v4.0/cli.js status
node v4.0/cli.js tools
```

## 📝 可用指令

| 指令 | 功能 | 場景 |
|------|------|------|
| `start` | 啟動機器人 | 私聊/群組 |
| `stop` | 停止機器人 | 私聊/群組 |
| `狀態` | 查看狀態 | 私聊/群組 |
| `選單` | 顯示選單 | 私聊/群組 |
| `天氣 [城市]` | 查詢天氣 | 私聊/群組 |

### 滑動回覆指令
滑動任意訊息並回覆：
- `start` - 以該訊息為上下文啟動
- `天氣` - 智能識別天氣查詢
