1
接收用戶訊息

client.on('message_create', async (msg) => { const chat = await msg.getChat(); const chatId = chat.id._serialized; const messageBody = msg.body.trim(); console.log(`[${getTimestamp()}] 📩 收到訊息 [聊天: ${chatId}]`); });
⬇️
2
檢測滑動回覆類型
判斷訊息是否為滑動回覆，以及回覆的對象是機器人還是其他人
if (msg.hasQuotedMsg) { const quoted = await msg.getQuotedMessage(); const isReplyToBot = quoted.fromMe; const isReplyToOther = !quoted.fromMe; if (isReplyToBot) testType = 1; // 滑動回覆機器人 else if (isReplyToOther) testType = 2; // 滑動回覆其他人 }
⬇️
3
🧠 智能指令優先處理（v4.0 新增）
在群組授權檢查之前，先處理天氣查詢等智能指令
這樣用戶可以直接輸入「今天天氣如何」而無需滑動回覆
// 智能指令解析 - 優先處理 const SmartCommand = require('./tools/smart-command'); const smartCommand = new SmartCommand({ verbose: false, chatId: chatId }); const parseResult = smartCommand.parse(messageBody); // 如果是天氣查詢指令，直接處理（無需滑動回覆） if (parseResult.matched && parseResult.command === 'weather') { console.log(`🧠 智能識別天氣查詢（無需滑動回覆）`); // 執行天氣查詢... return; }
⬇️
🔒 群組指令授權檢查
在群組中，只有以下情況才處理指令：
✅ 授權通過
• 滑動回覆機器人訊息
• start / stop 指令
• 天氣查詢（已在上方處理）
❌ 授權失敗
• 未滑動回覆機器人
• 忽略指令，不處理
• 返回，不執行後續邏輯
⬇️
4
處理滑動回覆指令
檢查滑動回覆的是哪個工具的訊息，直接啟動對應工具（無需輸入數字）
if (msg.hasQuotedMsg && quoted.fromMe) { const quotedMessageId = quoted.id._serialized; const toolName = chatManager.getToolByMessageId(quotedMessageId); // v4.0: 滑動工具訊息直接啟動工具（無需輸入數字） if (toolName) { if (toolName === '主頁') { await sendToolMenu(msg, chatId); // 返回主頁 } else if (toolName === '停止') { // 停止所有功能 } else { // 啟動對應工具 const pluginIndex = pluginManager.plugins.findIndex(p => p.name === toolName); const plugin = await pluginManager.activatePlugin(pluginIndex, chatId); } } }
⬇️
5
處理一般指令
處理 start、stop、狀態、選單等指令
// 處理 "start" 指令 - 顯示工具選單 if (messageBody.toLowerCase() === 'start') { await sendToolMenu(msg, chatId); // v4.0: 分開訊息發送 return; } // 處理 "stop" 指令 - 停止所有功能 if (messageBody.toLowerCase() === 'stop') { const stoppedPlugins = chatManager.stopAllPlugins(chatId); await msg.reply(`✅ 已停止工具：${stoppedPlugins.join(', ')}`); return; }
⬇️
6
智能指令解析（v3.0）
使用自然語言解析用戶意圖，自動轉換為系統指令
// 載入 SmartCommand 工具 const SmartCommand = require('./tools/smart-command'); const smartCommand = new SmartCommand({ verbose: false, chatId: chatId }); // 解析用戶輸入 const parseResult = smartCommand.parse(messageBody); if (parseResult.matched) { switch (parseResult.command) { case 'msg-extractor': /* 啟動記錄訊息 */ break; case 'auto-reply': /* 啟動自動回覆 */ break; case 'start': /* 顯示選單 */ break; case 'status': /* 顯示狀態 */ break; // case 'weather': /* 已在上方處理 */ } }
⬇️
7
工具訊息處理
將訊息傳遞給已啟動的工具處理
// 檢查這個聊天是否有活動的工具 if (chatManager.hasActivePlugins(chatId)) { const activePlugins = chatManager.getActivePlugins(chatId); for (const pluginName of activePlugins) { const pluginData = pluginManager.getPluginInstance(pluginName, chatId); if (pluginData && pluginData.instance && pluginData.instance.processMessage) { await pluginData.instance.processMessage(msg); } } }
⬇️
8
預設處理與回應
處理其他訊息或顯示提示
// 指令：!test（系統測試指令） if (messageBody === '!test') { await msg.reply('已連接，可互動'); return; } // 其他訊息 - 只顯示不處理 console.log(`[${getTimestamp()}] ℹ️ 訊息已顯示，等待後續處理 [聊天: ${chatId}]`);

📱 使用說明

🚀 快速開始
  • 輸入 "start" 開啟工具選單
  • 滑動工具訊息直接使用
  • 輸入 "stop" 停止所有功能
  • 輸入 "狀態" 查看運行中工具
🌤️ 天氣查詢
  • 「今天天氣如何」- 查詢天氣
  • 「香港天氣」- 指定城市
  • 「台北天氣」- 支援多城市
  • 無需滑動回覆即可查詢
👥 群組使用
  • 滑動回覆機器人訊息
  • 天氣查詢可直接輸入
  • start/stop 所有人可用
  • 其他指令需滑動回覆
💬 私聊使用
  • 直接輸入指令即可
  • 無需滑動回覆
  • 支援所有智能指令
  • 更簡便的操作方式
