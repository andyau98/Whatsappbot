# WhatsApp Bot v3.0 🤖

一個純本地的 WhatsApp 自動化機器人，支援智能指令解析，無需第三方服務。

## ✨ 主要功能

### 🧠 v3.0 智能指令解析（新增）
- 使用自然語言操作機器人，無需記憶固定指令
- 支援多種同義詞和表達方式
- 自動識別用戶意圖並執行對應操作
- 智能處理負面指令（「不要」「停止」）

### 💬 每個聊天獨立管理（v2.0）
- 每個聊天（私聊/群組）都有自己獨立的工具狀態
- 在一個聊天中啟動的工具，不影響其他聊天
- 支援多個聊天同時使用不同工具

### 🔧 工具系統
- 模組化設計，易於擴展
- 每個工具可獨立運行
- 支援動態載入和卸載

## 📋 使用方式

### 傳統指令

| 指令 | 功能 |
|------|------|
| `start` | 顯示工具選單 |
| `1` / `2` / `3` | 選擇工具 |
| `0` | 取消選擇 |
| `stop` | 停止當前聊天的工具 |
| `test` / `!test` | 測試連接 |

### 智能語言指令（v3.0 推薦）

直接說出你的需求：

#### 📝 記錄訊息
- "幫我記住這個"
- "開始記錄訊息"
- "保存這個對話"
- "我要儲存訊息"

#### 💬 自動回覆
- "開始自動回覆"
- "幫我回覆訊息"
- "啟動自動回答"

#### 🛑 停止功能
- "停止所有功能"
- "關閉所有工具"
- "不要記錄了"
- "結束運作"

#### � 其他
- "顯示選單" - 查看所有功能
- "有什麼功能" - 查看可用工具
- "查看狀態" - 了解目前運作情況

## �🚀 快速開始

### 1. 安裝依賴

```bash
npm install
```

### 2. 啟動程式

```bash
node index.js
```

### 3. 首次登入

1. Terminal 會顯示 QR Code
2. 打開手機 WhatsApp → 設定 → 已連結裝置 → 連結裝置
3. 掃描 QR Code 完成登入
4. 登入憑證會自動儲存，下次啟動無需重新掃描

### 4. 測試功能

**智能指令測試：**
- 發送「幫我記住這個」→ 啟動訊息記錄
- 發送「顯示選單」→ 查看所有工具

**傳統指令測試：**
- 私聊：直接發送 `start`
- 群組：@機器人名稱 + `start`

## 🔧 工具說明

### 1. auto-reply（自動回覆器）
根據關鍵字自動回覆訊息，支援自定義回覆規則。

**特點：**
- ✅ 支援多個關鍵字匹配
- ✅ 支援精確匹配和包含匹配
- ✅ 可區分私聊和群組
- ✅ 統計回覆數據

### 2. msg-extractor（訊息提取器）
從 WhatsApp 提取訊息並儲存到本地電腦。

**特點：**
- ✅ 自動儲存所有收到的訊息
- ✅ 分類儲存文字訊息和媒體檔案
- ✅ 按日期整理（每天一個 JSON 檔案）
- ✅ 支援每個聊天獨立儲存

### 3. smart-command（智能指令解析器）
使用自然語言解析用戶意圖，自動轉換為系統指令。

**特點：**
- ✅ 自然語言理解
- ✅ 多種表達方式支援
- ✅ 智能意圖識別
- ✅ 信心度評分
- ✅ 負面指令處理

## ⚠️ 注意事項

1. **首次登入**需要掃描 QR Code
2. **群組功能**需要 @提及才能觸發回覆
3. **憑證儲存**在 `.wwebjs_auth` 資料夾，請勿刪除（除非需要重新登入）
4. **純本地運行**，無需網絡穿透或第三方服務
5. **智能指令**需要滿足信心度門檻才會觸發
