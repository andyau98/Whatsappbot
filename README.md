# WhatsApp Bot 專案 🤖

最後更新：2026-04-03

## � 專案結構

```
Whatsappbot/
├── index.js              # 主程式入口（目前版本）
├── tools/                # 工具目錄
│   ├── msg-extractor/    # 訊息提取工具
│   │   ├── r1/          # v1-v3 使用
│   │   └── r2/          # v4 使用
│   ├── smart-command/    # 智能指令解析
│   │   ├── r1/
│   │   └── r2/
│   └── weather/          # 天氣查詢
│       ├── r1/
│       └── r2/
├── v1.0/                 # 版本 1.0
├── v2.0/                 # 版本 2.0
├── v3.0/                 # 版本 3.0
├── v4.0/                 # 版本 4.0（最新）
└── trynerror/            # 測試/實驗版本
```

## � 版本說明

| 版本 | 主要功能 | 工具版本 |
|------|----------|----------|
| v1.0 | 基礎功能 | - |
| v2.0 | 聊天獨立管理 | - |
| v3.0 | 智能指令解析 | tools/r1 |
| v4.0 | 滑動回覆、CLI 管理 | tools/r2 |

## 🛠️ 工具版本管理

每個工具支援多版本（r1, r2, r3...）：
- **r1**: 基礎版本（v1-v3 使用）
- **r2**: 增強版本（v4 使用）
- **r3+**: 未來版本

更新工具時：
1. 複製最新版本（如 r2 → r3）
2. 在 r3 修改功能
3. 更新主程式引用 r3

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 運行目前版本
node index.js

# 或運行特定版本
node v4.0/index.js
```

## 📝 可用指令

| 指令 | 功能 |
|------|------|
| `start` | 顯示工具選單 |
| `stop` | 停止工具 |
| `狀態` | 顯示狀態 |
| `選單` | 顯示選單 |

## 🔧 工具清單

- **weather**: 天氣查詢（wttr.in API）
- **smart-command**: 智能指令解析
- **msg-extractor**: 訊息提取
