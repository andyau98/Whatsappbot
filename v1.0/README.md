# WhatsApp 本地機器人

一個純本地的 WhatsApp 自動回覆機器人，無需第三方服務（如 Twilio、Ngrok）。

## 📋 功能規格

### 訊息處理規則

| 場景 | 處理方式 |
|------|----------|
| **私聊訊息** | 所有指令直接回覆，無需 @提及 |
| **群組訊息** | 需同時滿足：<br>1. 有 @提及（msg.mentionedIds 非空）<br>2. 訊息包含指令（!test 或 test）<br>否則只在 Terminal 顯示，不回覆 |

### 支援指令

| 指令 | 回覆內容 | 說明 |
|------|----------|------|
| `!test` | 已連接，可互動 | 測試連接狀態 |
| `test` / `Test` / `TEST` | it's ok | 不分大小寫 |

## 🚀 快速開始

### 1. 安裝依賴

```bash
npm install
```

### 2. 啟動程式

```bash
node index.js
```

### 3. 首次登入

1. Terminal 會顯示 QR Code
2. 打開手機 WhatsApp → 設定 → 已連結裝置 → 連結裝置
3. 掃描 QR Code 完成登入
4. 登入憑證會自動儲存，下次啟動無需重新掃描

### 4. 測試功能

**私聊測試：**
- 直接發送 `!test` 或 `test`

**群組測試：**
- @機器人名稱 + 指令，例如：「@機器人 test」

## 📁 專案結構

```
Whatsappbot/
├── index.js          # 主程式
├── package.json      # 專案配置
├── .wwebjs_auth/     # 登入憑證（自動生成）
└── README.md         # 本文件
```

## 🔧 常用操作

### 重置 QR Code（重新登入）

```bash
rm -rf .wwebjs_auth
node index.js
```

### 停止程式

按 `Ctrl + C` 停止程式

## 📝 記錄功能

- **所有訊息**都會顯示在 Terminal
- 顯示內容包括：時間、類型（私聊/群組）、發送者、內容
- 符合條件的訊息會回覆，其他只顯示

## ⚠️ 注意事項

1. **首次登入**需要掃描 QR Code
2. **群組功能**需要 @提及才能觸發回覆
3. **憑證儲存**在 `.wwebjs_auth` 資料夾，請勿刪除（除非需要重新登入）
4. **純本地運行**，無需網絡穿透或第三方服務
